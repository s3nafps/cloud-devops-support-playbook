# Cloud/DevOps Support Career Playbook

Concrete assets for a 90-day move toward:

- `Cloud Support Engineer`
- `DevOps Support Engineer`
- `Linux/Application Support Engineer`

This repo is not a SaaS idea and not a generic advice dump. It is an execution pack built around low-cost tools, public proof, and repeatable job-search execution.

## Who This Is For

- Support-oriented cloud/devops job search
- Evidence-first positioning from Algeria for remote or EMEA opportunities
- Limited-budget execution using GitHub, Markdown, CSV trackers, and existing public proof

## Publish-First Priority

If only the first 14 days matter, do these first:

1. Publish CloudOps Rescue Kit and case-study evidence.
2. Update LinkedIn using the provided copy.
3. Tailor the CV around Linux, Docker, monitoring, backup, troubleshooting, and support communication.
4. Track applications and buyer conversations every day.

## Operating Constraints

- Based in Algeria.
- Current income target is above `80,000 DA/month`.
- Budget is limited, so the default toolset is low-cost or free.
- Do not ask buyers to share passwords or private keys in chat.
- Do not build a SaaS until repeated demand is visible through calls, replies, or paid work.

## Recommended Workflow

1. Use the CV and LinkedIn docs to reposition public profiles.
2. Use CloudOps Rescue Kit as proof when applying.
3. Apply daily using the role filters and tracker.
4. Use outreach scripts only when you want to test buyer language or direct networking.
5. Record every signal before building anything bigger.
